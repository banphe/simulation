# Simulation

## Wymagania

- [Tailwind CSS CLI](https://tailwindcss.com/blog/standalone-cli) - pobierz `tailwindcss-windows-x64.exe` i umieść w `build/` jako `tailwindcss.exe`

## Uruchomienie

1. Pobierz repozytorium
2. Pobierz Tailwind CSS CLI i umieść w `build/tailwindcss.exe`
3. Uruchom `scripts\watch-tailwind.bat`
4. Otwórz `index.html` w przeglądarce

## Struktura

- `src/` - kod aplikacji
- `build/` - konfiguracja Tailwind i DaisyUI
- `scripts/` - skrypty pomocnicze
