# Simulation

Projekt symulacji finansowej z interfejsem MVP.

## Wymagania

- Tailwind CSS standalone binary (build/tailwindcss.exe) - [pobierz](https://github.com/tailwindlabs/tailwindcss/releases)

## Uruchomienie

1. Umieść `tailwindcss.exe` w folderze `build/`
2. Uruchom `scripts/watch.bat` do kompilacji CSS
3. Otwórz `index.html` w przeglądarce

## Struktura

- `src/` - kod źródłowy (MVP: Model-View-Presenter)
- `build/` - konfiguracja budowania
- `scripts/` - skrypty pomocnicze
